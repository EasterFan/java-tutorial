      # Zipkin

      Zipkin Тў»СИђСИфтЪ║С║ј Java т╝ђтЈЉуџётѕєтИЃт╝ЈУиЪУИфу│╗у╗Ъсђѓт«ЃжЄЄжЏєТюЅтіЕС║јУДБтє│ТюЇтіАТъХТъёСИГт╗ХУ┐ЪжЌ«жбўуџёт«ъТЌХТЋ░ТЇ«сђѓтЁХТаИт┐ЃтіЪУЃйТў»жЄЄжЏєтњїТЪЦТЅЙТЋ░ТЇ«сђѓ

      # уЅ╣ТђД

      тдѓТъюТЌЦт┐ЌТќЄС╗ХСИГТюЅУиЪУИф ID№╝їтѕЎтЈ»С╗ЦуЏ┤ТјЦУи│УЄ│У»ЦУиЪУИф IDсђѓ тљдтѕЎ№╝їТѓетЈ»С╗ЦтЪ║С║јт▒ъТђДУ┐ЏУАїТЪЦУ»б№╝їСЙІтдѓТюЇтіА№╝їТЊЇСйютљЇуД░№╝їТаЄуГЙтњїТїЂу╗ГТЌХжЌ┤сђѓ т░єСИ║ТѓеТђ╗у╗ЊСИђС║ЏТюЅУХБуџёТЋ░ТЇ«№╝їСЙІтдѓтюеТюЇтіАСИГУі▒У┤╣уџёТЌХжЌ┤уЎЙтѕєТ»ћС╗ЦтЈіТЊЇСйюТў»тљдтц▒У┤Цсђѓ

      Zipkin UI У┐ўТЈљСЙЏС║єСИђСИфСЙЮУхќтЁ│у│╗тЏЙ№╝їУ»ЦтЁ│у│╗тЏЙТўЙуц║С║єТ»ЈСИфт║ћућеуеІт║ЈСИГУиЪУИфС║єтцџт░ЉСИфУ»иТ▒ѓсђѓУ┐Ўт»╣С║јУ»єтѕФУЂџтљѕУАїСИ║№╝ѕтїЁТІгжћЎУ»»Уи»тЙёТѕќт»╣СИЇУхъТѕљСй┐ућеуџёТюЇтіАуџёУ░Ѓуће№╝ЅтЙѕТюЅтИ«тіЕсђѓ

      Tracer тГўтюеС║јСйауџёт║ћућеСИГ№╝їт«ЃУ┤ЪУ┤БжЄЄжЏєтЁ│С║јти▓тЈЉућЪТЊЇСйюуџёт«ъТЌХтЁЃТЋ░ТЇ«сђѓт«ЃС╗гжђџтИИС╝џТБђТхІт║Њ№╝їтЏаТГцт»╣С║јућеТѕиТў»жђЈТўјуџёсђѓСЙІтдѓ№╝їти▓ТБђТхІуџё Web ТюЇтіАтЎеУ«░тйЋт«ЃСйЋТЌХТјЦТћХтѕ░У»иТ▒ѓ№╝їС╗ЦтЈіСйЋТЌХтЈЉжђЂтЊЇт║ћсђѓТћХжЏєуџёУиЪУИфТЋ░ТЇ«уД░СИ║Уиет║д№╝ѕSpan№╝Ѕсђѓ

      Instrumentation С┐ЮУ»ЂС║єућЪС║Дуј»тбЃуџёт«ЅтЁеТђДтњїтЙѕт░Љуџёт╝ђжћђсђѓтЏаТГц№╝їт«ЃС╗гС╗ЁтюетєЁжЃеС╝аТњГ ID№╝їС╗ЦтЉіуЪЦТјЦТћХТќ╣ТГБтюеУ┐ЏУАїУ┐йУИфсђѓт«їТѕљуџё Span т░єжђџУ┐ЄтцќжЃежђџС┐АтЉіуЪЦ Zipkin№╝їу▒╗С╝╝С║јт║ћућеуеІт║Јт╝ѓТГЦТіЦтЉіТїЄТаЄуџёТќ╣т╝Јсђѓ

      СЙІтдѓ№╝їтйЊУиЪУИфТЪљСИфТЊЇСйют╣ХСИћжюђУдЂтЈЉтЄ║ http У»иТ▒ѓТЌХ№╝їС╝џТи╗тіаСИђС║Џ header ТЮЦС╝аТњГ IDсђѓheader СИЇућеС║јтЈЉжђЂУ»ду╗єС┐АТЂ»№╝їСЙІтдѓТЊЇСйютљЇуД░сђѓ

      УЃйтцЪт░єТЋ░ТЇ«тЈЉжђЂтѕ░ Zipkin уџёТБђТхІт║ћућеуеІт║ЈСИГуџёу╗ёС╗Х№╝їУбФуД░СИ║ ReporterсђѓReporter ТюЅтцџуДЇС╝аУЙЊТќ╣т╝Ј№╝їтЈ»С╗Цт░єУиЪУИфТЋ░ТЇ«тЈЉжђЂтѕ░ Zipkin жЄЄжЏєтЎе№╝їтљјУђЁт░єУиЪУИфТЋ░ТЇ«ТїЂС╣ЁтїќС┐ЮтГўтѕ░тГўтѓеСИГсђѓуеЇтљј№╝їAPI С╝џТЪЦУ»бтГўтѓеС╗ЦтљЉ UI ТЈљСЙЏТИ▓ТЪЊТЋ░ТЇ«сђѓ
